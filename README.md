# 迷你游戏合集 🎮

一个纯前端实现的迷你游戏合集，包含贪吃蛇、打砖块、俄罗斯方块等经典游戏。

## 🗂️ 项目结构

```
迷你游戏合集/
├── index.html              # 游戏主页面，游戏选择界面
├── start.sh                # 启动脚本
├── README.md               # 说明文档
├── LICENSE                 # MIT许可证文件
├── css/
│   ├── style.css           # 主页面样式
│   └── games.css           # 游戏通用样式
├── js/
│   ├── storage.js          # 排行榜存储（localStorage）
│   ├── snake.js            # 贪吃蛇游戏逻辑
│   ├── brick.js            # 打砖块游戏逻辑（开发中）
│   ├── tetris.js           # 俄罗斯方块游戏逻辑
│   └── minesweeper.js      # 扫雷游戏逻辑
└── games/
    ├── snake.html          # 贪吃蛇游戏页面
    ├── brick.html          # 打砖块游戏页面（开发中）
    ├── tetris.html         # 俄罗斯方块游戏页面
    └── minesweeper.html    # 扫雷游戏页面
```

## 🎮 当前可用游戏

### ✅ 贪吃蛇游戏
- **文件**: `games/snake.html`
- **控制**: 键盘方向键 ↑ ↓ ← →
- **特点**: 
  - 实时显示当前得分和最高分
  - 记录蛇的长度
  - 排行榜系统
  - 暂停/继续功能

### ✅ 俄罗斯方块游戏
- **文件**: `games/tetris.html`
- **控制**: 键盘方向键 ↑ ↓ ← →
- **特点**: 
  - 经典俄罗斯方块玩法
  - 多种方块类型（I、O、T、L、J、S、Z）
  - 等级提升和速度增加
  - 排行榜系统

### ✅ 扫雷游戏
- **文件**: `games/minesweeper.html`
- **控制**: 鼠标左键（揭开）/ 右键（标记）
- **特点**: 
  - 三种难度级别
  - 实时计时
  - 排行榜系统
  - 经典扫雷玩法

### ✅ 五子棋游戏
- **文件**: `games/gomoku.html`
- **控制**: 鼠标点击
- **特点**: 
  - 人机对战
  - AI拟人化对话
  - 基于对弈数据的AI训练
  - 存储空间管理

## 🚀 使用方法

### 方法一：启动脚本
```bash
bash start.sh
```
或双击 `start.sh` 文件（macOS/Linux）

### 方法二：直接打开
- 用浏览器直接打开 `index.html` 文件
- 或访问游戏页面：
  - 贪吃蛇: `games/snake.html`

### 方法三：部署到服务器
1. 将整个目录上传到Web服务器
2. 访问 `index.html`

## 🎯 游戏控制

### 贪吃蛇
- **↑ ↓ ← →**: 控制蛇的移动方向
- **空格键**: 开始游戏/暂停/继续
- **吃到食物**: 得分+10，蛇身变长
- **游戏结束**: 撞墙或撞到自己

## 🏆 排行榜系统

- 自动保存每个游戏的最高分
- 记录最后游玩时间
- 显示游戏历史统计

## 💡 技术特点

- **纯前端实现**: HTML5 + CSS3 + JavaScript
- **响应式设计**: 支持手机和电脑
- **本地存储**: 使用 localStorage 保存游戏数据
- **模块化结构**: 易于扩展新游戏
- **无外部依赖**: 不需要任何框架或库

## 🔧 自定义选项

### 修改游戏配置
在 `js/snake.js` 中可以修改：
- `gridSize`: 网格大小（默认20）
- `canvas.width/height`: 画布尺寸（默认400x400）
- 颜色样式在 `css/games.css` 中修改

### 添加新游戏
1. 在 `games/` 目录创建游戏页面
2. 在 `js/` 目录添加游戏逻辑
3. 在 `index.html` 中添加游戏卡片
4. 在 `storage.js` 中注册游戏名称

## 📱 移动端支持

- 自适应屏幕大小
- 触摸操作支持
- 响应式布局

## 🎉 开始游玩

现在打开 `index.html` 或运行 `start.sh` 开始您的游戏之旅！
